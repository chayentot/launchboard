# LaunchBoard v6 — Mobile Commerce Experience

LaunchBoard v6 keeps the existing desktop website and upgrades the cellphone experience.

## Mobile flow

- **Home** — Discover Products, Recommended Products, Featured Creators
- **Sell** — opens Publish a Product directly
- **Messages** — opens conversations
- **Profile** — opens My Dashboard
- Dashboard avatar menu — Edit Creator Profile, View Public Profile, Log out

## Mobile design improvements

- Compact sticky header
- Small search field
- Quick links for Discover, For You, and Creators
- Two-column product grid
- Smaller product titles and cards
- Compact creator rows
- Compact dashboard statistics and forms
- Smaller product detail title area
- Four-item bottom navigation
- Android Back on inner pages returns to Home instead of closing immediately

## Installation

Upload every file in this package to the same location as the current LaunchBoard project.
After GitHub Pages finishes deploying, clear the Android app/WebView cache or reinstall the test APK.
