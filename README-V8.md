# LaunchBoard v8 — Mobile Marketplace

## Home
- New Today horizontal rail
- Trending horizontal rail
- Recommendations and Featured Creators remain
- Marketplace-style product cards
- Product loading skeletons

## Product view
- Like, Share, and Visit actions under the image
- Compact creator profile and Message action
- Related Products
- Sticky Message and Visit Product controls on mobile

## Dashboard
- Compact Products, Views, Likes, and Followers statistics

## Notifications
- All, Unread, Messages, Social, and Products filters

## Navigation
Home | Message | Sell | Notification | Profile

## Installation
Upload every file in this package. The files use the `8.0.0` cache version.
Clear Android app data after the GitHub Pages deployment succeeds.

The existing v6.5.1 messaging RLS and v7.4.1 notifications SQL must already be applied.
