# LaunchBoard V8.1.8 Patch

## Fixes
- Review submission now reliably inserts or updates one review per user and shows success/error feedback.
- The + button in chat now opens an attachment menu.
- Supports JPG, PNG, WEBP, PDF, DOC/DOCX, XLS/XLSX and TXT up to 15 MB.
- Images preview inside chat; documents display as downloadable file cards.

## Upload these files
- product.html
- product.js
- messages.html
- messages.js
- styles.css

## Database
Run `supabase-v8.1.8-reviews-attachments.sql` first in Supabase SQL Editor.
