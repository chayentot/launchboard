# LaunchBoard v6.1 — Mobile Polish

This update keeps the v6 layout and applies the requested refinements.

## Changes

- Removed the duplicate search field from Discover.
- Kept only the top product/creator search.
- Filters and Sort remain available.
- Removed the Explore LaunchBoard label above Discover Products.
- Reduced hamburger button size.
- Removed Add Product from the dashboard header.
- Moved Add Product beside Your Products.
- Dashboard header now shows only My Dashboard and the profile avatar.
- Tightened product cards, dashboard rows, and section spacing.
- Desktop styling remains unchanged.

## Install

Upload all files in this package, or at minimum:

- index.html
- dashboard.html
- styles.css

Then clear the Android app/WebView cache before testing.
