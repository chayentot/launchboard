# LaunchBoard V4.3 Launch Readiness

## Install

Run `supabase-v4.3-launch-readiness.sql` in a new Supabase SQL query.

Replace all files in this package except your working `config.js`.

In Supabase Authentication URL settings, add:

`https://chayentot.github.io/launchboard/reset-password.html`

to the allowed redirect URLs.

## Added

- Report product and creator buttons
- Administrator reports queue
- Password-reset email and new-password page
- Terms, Privacy, Community Guidelines, Prohibited Products, and Support pages
- External website warning
