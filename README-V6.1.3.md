# LaunchBoard v6.1.3 — Left Alignment and Creator Grid

## Changes

- Discover Products heading is forced to the left.
- Featured Creators heading is forced to the left.
- Featured Creators now displays 3 creator cards in one row on mobile.
- Creator cards are compact and centered for narrow phone screens.
- Desktop layout remains unchanged.

## Install

Replace:
- index.html
- styles.css

Then clear the Android app/WebView cache and reopen the app.
